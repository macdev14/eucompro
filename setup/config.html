If you have not already done so:

  1. Find the config.example.php file in the base folder (or src/ folder).

    Note: There are 2 files that start with config, be sure to use the example one.
  2. Rename the file to config.php
  3. Enter your database connection information (MySQL settings) into the config.php file at this time. Also, ensure that all of the appropriate directories have the correct permissions, as noted in the installation instructions found in the user manual.

For more information, see the installation instructions.

The Wizard will now check the DB settings you have entered into the config.php file.
